AIDA Django server ishlayapti

Backend muvaffaqiyatli ko'tarildi. Frontend build hali yaratilmagan, shu sabab Django vaqtincha ushbu sahifani ko'rsatyapti.

Build tayyor bo'lgach, Django avtomatik ravishda dist/index.html va dist/assets ni serve qiladi.

Asosiy buyruqlar: .venv\Scripts\python manage.py runserver va frontend uchun npm run build.
